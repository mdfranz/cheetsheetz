# Installing Fonts 

Create `~/.local/share/fonts` and download from https://www.nerdfonts.com/font-downloads

Copy *.otf and *.ttf into that directory and run `fc-cache -fv`


# Keyboard Shortcuts

- `[space] - e` - toggle file manager
- `[space] - t, f` - bring up floating shell
