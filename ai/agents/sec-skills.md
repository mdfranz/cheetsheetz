# Projects
- [AgentSecOps/SecOpsAgentKit](https://github.com/AgentSecOps/SecOpsAgentKit)
  - A security operations toolkit for AI coding agents (Claude Code) with 25+ skills for vulnerability detection, container scanning, secret identification, and policy enforcement.
- [mdfranz/sec-skillz](https://github.com/mdfranz/sec-skillz)
  - A collection of specialized agent skills for security analysis (Gemini, Claude, Codex), including Suricata EVE JSON and AWS CloudTrail log analysis.
- [TransilienceAI Community Tools](https://github.com/transilienceai/communitytools)
  - Open-source Claude Code skills, agents, and slash commands for AI-powered penetration testing, bug bounty hunting, and security research.
- [Ghost Security Skills](https://github.com/ghostsecurity/skills)
  - A marketplace of AI-native skills that turns Claude Code into an application security expert, providing comprehensive code security analysis, dependency/secrets scanning, and dynamic vulnerability testing.
