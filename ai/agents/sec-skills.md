# Projects
- [AgentSecOps/SecOpsAgentKit](https://github.com/AgentSecOps/SecOpsAgentKit) - A security operations toolkit for AI coding agents (Claude Code) with 25+ skills for vulnerability detection, container scanning, secret identification, and policy enforcement.
- [mdfranz/sec-skillz](https://github.com/mdfranz/sec-skillz) - A collection of specialized agent skills for security analysis (Gemini, Claude, Codex), including Suricata EVE JSON and AWS CloudTrail log analysis.
- [TransilienceAI Community Tools](https://github.com/transilienceai/communitytools) - Open-source Claude Code skills, agents, and slash commands for AI-powered penetration testing, bug bounty hunting, and security research.
- [Ghost Security Skills](https://github.com/ghostsecurity/skills) - A marketplace of AI-native skills that turns Claude Code into an application security expert, providing comprehensive code security analysis, dependency/secrets scanning, and dynamic vulnerability testing.

# Google Security (mostly dandye)
- [dandye/agentic_soc_agentspace](https://github.com/dandye/agentic_soc_agentspace) - Demonstrates deploying security-focused AI agents to Google Cloud's Vertex AI Agent Engine, integrating with security tools via the Model Context Protocol (MCP).
- [dandye/ai-runbooks](https://github.com/dandye/ai-runbooks) - Agentic SOC Runbooks for AI-assisted cybersecurity workflows. It standardizes incident response and threat analysis procedures across AI platforms like Claude, Gemini, and Cline.
